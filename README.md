# NodeJS Labs

This repository contains Node.js and Express.js lab exercises and demo projects used for learning and practice.

## Prerequisites
- Node.js v14 or later
- npm (included with Node.js)

## Quick start
1. Open a terminal and navigate to the folder you want to run, for example:

```bash
cd "day 1"
```

2. Install dependencies (if the folder has a `package.json`):

```bash
npm install
```

3. Run the example (common entry files are `index.js` or `server.js`):

```bash
node index.js
# or
node server.js
```

## Project structure (top-level folders)
- `day 1/` - basic Node.js examples and file exercises
- `Demo_ExpressJS/` - simple Express demo with controllers, routes, and middlewares
- `Downloaded_Lab/` - additional lab material (Day 2 folder included)
- `Lab_ExpressJS/` - Express lab with controllers, routes, and services

## Notable folders
- `controllers/` - request handlers
- `routes/` - route definitions
- `middlewares/` - Express middleware examples
- `services/` - business logic and data access

## Notes
- Many subfolders include their own `package.json` and examples. Run `npm install` inside the folder before starting.
- If you want, I can add individual README files per lab subfolder summarizing how to run each one.
